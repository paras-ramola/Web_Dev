*HTML
Hyper Text Markup Langauge
Html is the code that is used to structure a web page and it's content.
The component used to design the structure of websites are called HTML tags.

*HTML tags
A container for some container or other Html tags

eg: <p>These are  paragraph tag</p>
content is written between these two tags
tags+content=Element

*Anchor tag
Used to add links to your page
syntax:<a href="site link">Display name</a>
eg:<a href="https://www.google.com/">Google</a>

*Two types of link
1.Absolute link:Link of another website like google,tesla,etc.
2.Relative link:Link of our own webpage/txt in our folder.
eg:<a href="/folder path">Folder</a>

*Image tags
1.used to add images to your page
2.syntax:<img src="/image.png" alt="Random Image"/>
3.alt :gives a name for you image ,incase if the image link is not working
4.you can use both absolute or Relative link for your image
5.You can resize your image using attributes hieght and width.
